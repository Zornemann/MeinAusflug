# iOS / iPhone

Dieses Projekt ist für eine **Home-Screen-PWA** vorbereitet.

## Nutzung auf iPhone
1. Safari öffnen
2. Deine veröffentlichte App aufrufen
3. `Teilen` → `Zum Home-Bildschirm`
4. Danach startet die App im Standalone-Modus

## App Store
Für den Apple App Store reicht ein reiner Web-Wrapper oft nicht aus. Empfohlen ist zunächst die PWA-Variante.
